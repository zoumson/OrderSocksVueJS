This project uses VueJS as a framework
The built website is about purchasing socks online.
There are two types available: the green and the blue one. 
Hovering on colors above the add chart button will display the corresponding socks with specified color.
Green socks are in stock, adding them on chart increase the counter of items in the chart.
However the blue socks are out of stock, the add chart button is disable for that item.
This project also provide information on shipping fees and product details.
At the bottom of the website page, there is a section for review.
User can see previous reviews and add new reviews according to his or her experience.
